# FitTrack – Fitness Tracking Web Application

**FitTrack** is a full-stack fitness tracking web application designed to help users stay consistent and motivated in their health journey.  
It combines **personalized workout planning, habit tracking, and progress visualization** into a clean and user-friendly experience.

# 🚀 Key Features

### Personalized Workout Planner
Automatically generates **weekly workout plans** based on the user's selected goal:
- Fat Loss
- Muscle Gain
- General Fitness  

Each plan includes **daily exercises with sets and repetitions**.

### Workout Calendar with Streak Tracker
Track completed workouts through an **interactive calendar** that highlights:
- Workout streaks
- Monthly performance
- Completed workout days

### Weekly Habit Checklist
Encourages healthy routines such as:
- Drinking enough water
- Getting proper sleep
- Daily activity  

Users can view **weekly completion summaries** for better consistency.

### Dashboard Overview
A centralized dashboard showing:
- Workout completion stats
- Habit tracking progress
- Current workout streak

### Exercise Tutorials
Includes **exercise guides and video demonstrations** to help users maintain correct form and technique.

# 🛠️ Technology Stack

## Frontend
- **React** – Dynamic user interface
- **React Router** – Client-side routing
- **Tailwind CSS** – Modern and responsive styling

## Backend
- **Node.js** – Server-side runtime environment
- **Express.js** – Backend API framework

## Database
- **MongoDB** – NoSQL database
- **Mongoose** – MongoDB object modeling

## Authentication
- **JWT (JSON Web Token)** – Secure login and protected routes

# 🏁 Getting Started

Follow these steps to run **FitTrack locally**.

---

## 📌 Prerequisites

Make sure you have installed:

- **Node.js (v16 or higher)**
- **MongoDB** (local or MongoDB Atlas)
- **npm or yarn**
- **Git**

---

# ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/desaishro/fittrack.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd fittrack
```

### 3️⃣ Install Dependencies

Install dependencies in both client and server folders.

```bash
cd client
npm install
```

```bash
cd ../server
npm install
```

---

# ▶️ Running the Application

### Start Backend Server

```bash
cd server
npm run dev
```

### Start Frontend Client

```bash
cd client
npm start
```

The application should now run locally on:

```
http://localhost:3000
```

# 💡 Usage

1. Register or log in to your account.
2. Choose your fitness goal.
3. View your personalized weekly workout plan.
4. Track daily workouts using the calendar.
5. Complete daily habits to maintain streaks.
6. Monitor your progress through the dashboard.
